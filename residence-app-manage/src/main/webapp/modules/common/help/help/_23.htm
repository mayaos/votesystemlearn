4.3.4 终端设备管理

点击菜单中终端设备管理，页面如图4.3.22所示，这个页面默认列出该账户下所有的设备数据，通过本页面可以实现对设备的添加、修改、删除、软件升级、查找操作及查找框的清空操作。页面的A区域为查询条件，B区域为功能项以及数据的展示。

图4.3.22 终端设备管理页面

 

具体的功能介绍：

1、设备添加

    添加主控器：点击菜单栏中的，在终端设备管理页面中弹出如图4.3.23所示的主控器的设备编辑界面，选择安装位置并输入主控器唯一的标识码，当标识码重复添加时，将弹出“设备标识码已存在，请检查！”的提示框。

图4.3.23 添加主控器的设备编辑界面

   

添加门锁终端：门锁终端安装在主控器下。选中某一主控器，右键点击，弹出如图4.3.24所示的门锁终端的设备编辑界面，输入门锁唯一的标识码，当标识码重复添加时，将弹出“设备标识码或内部标识码已存在，请检查！”的提示框。

图4.3.24 添加门锁终端的设备编辑界面

 

2、设备编辑

编辑主控器：选中某一主控器，右键点击，弹出如图4.3.25所示的界面，修改对应信息点击保存即可。

图4.3.25 编辑主控器的设备编辑界面

 

编辑门锁终端：选中某一门锁，右键点击，弹出如图4.3.26所示的界面，修改对应信息点击保存即可。

图4.3.26 编辑门锁终端的设备编辑界面

 

3、设备删除

删除主控器：选中某一主控器，右键点击，当主控器下有配置时，会弹出如图4.3.27的提示框，若删除则点击确定；当主控器下无配置时，会弹出“您确定要删除吗？”的提示框，若删除则点击确定。

图4.3.27 提示框

 

删除门锁终端：选中某一门锁终端，右键点击，当门锁终端下有出入卡权限时，会弹出如图4.3.28的提示框，若删除则点击确定；当门锁终端下无出入卡权限时，会弹出“您确定要删除吗？”的提示框，若删除则点击确定。

图4.3.28 提示框

 

4、关于主控器的其他操作

下载配置：选中某一主控器，右键点击，则弹出“命令已在排队中！”的提示框，等待后台处理。

清除配置：选中某一主控器，右键点击，则弹出“命令已在排队中！”的提示框，等待后台处理。

重启主控器：选中某一主控器，右键点击，则弹出“命令已在排队中！”的提示框，等待后台处理。

    5、关于门锁终端的其他操作

清空所有权限：选中某一门锁终端，右键点击，则弹出“命令已在排队中！”的提示框，等待后台处理。

重启门锁终端：选中某一门锁终端，右键点击，则弹出“命令已在排队中！”的提示框，等待后台处理。

    6、软件升级

选择要升级的设备后，点击会弹出编辑框，如下图所示，在这里选择升级文件名称后，点击确定即可。

图4.3.29 软件升级

  

    7、查找

    在查找框中输入查找条件，点击，页面B区域中则会显示对应条件的设备信息。

8、查询条件清空

在终端设备管理页面，点击，可清空填写过的查询条件。
