2.2 系统功能简介

本系统主要实现的功能包括：

    1、基础信息管理：人员信息（租客、房东、家属、巡防人员）、居所信息（房屋以及房屋所属区域）、终端信息（主控器、门锁）的录入、修改、删除、查询功能。

    2、卡管理：门禁SAM卡、发卡SAM卡、出入卡的发卡、注销、修改，出入卡的挂失、解挂、补卡、修改；卡权限管理，通过人员权限和门锁权限实现卡片开门权限的添加、更新、清除、删除。

    3、动态信息：住户出入记录上传，行为轨迹跟踪；安防告警信息监控，全方位监控设备情况和门锁安全隐患预警。

    4、布防管理：系统导入具有犯罪史或者有犯罪嫌疑的人员，通过事前布控（当录入已布控的人员时，系统会提示预警）；事后布控（通过监控布控人员行为轨迹，跟踪其行踪，为侦查破案提供证据）。

    5、数据统计分析：按流动人口年龄分布、来源地统计；房屋建筑类型、出租情况统计；刷卡次数按时间统计等。通过数据统计，定位其发展趋势，为政府决策提供依据。

    6、系统维护：账号管理、角色设置、桌面设置、菜单设置、日志管理。

通过角色设置，实现各级区域操作员权限的归类划分，而账号管理则是角色的具体化，它不仅拥有角色，还拥有用户名、密码、联系方式、区域等信息，可以通过权限定制或屏蔽实现某些操作功能项的操作权限。
