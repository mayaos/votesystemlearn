5.1 发卡插件和驱动安装

    发卡插件：无线发卡组件(DeliverCardCOM_Setup) 和USB插件(SDI01x_generic_win_driver_V5.24_V5.35)。点击压缩包，右击解压压缩包，文件就从压缩包解压出来。

 

1、无线发卡组件安装

（1）、双击DeliverCardCOM_Setup图标，进行安装无线安全发卡组件安装程序，出现如图页面

 

    （2）、双击后弹出欢迎使用“无线安全发卡组件”安装向导如下如所示，点击安装

 

    （3）、点击安装后，将自动进行程序的安装，进入完成界面，点击完成即可。

 

    2、发卡器USB驱动安装

（1）、首先将读卡器的USB连接电脑，发卡器指示灯亮起黄色灯进入准备状态；将卡放置发卡器上，若发卡器指示灯红黄闪烁，则可以正常发卡，否则需要查看驱动是否操作正确。

（2）、右击桌面上我的电脑，点击属性弹出系统属性对话框，点击硬件进入设备管理界面如下图所示：

 

（3）、在点击设备管理器按钮，进入设备管理器对话框，打开智能卡阅读器，选择第一个右击，在出现的对话框中点击更新驱动程序，如图

 

（4）、在弹出的硬件更新向导的对话框中选择从列表或制定位置安装（高级），点击下一步，如图

 

    （5）、在弹出的对话框中，选择第一个在这些位置上搜索最佳的驱动程序，勾选在搜索中包括的位置，点击浏览按钮，找到之间解压的智能卡USB驱动，点击下一步，如图

 

    （6）、点击下一步，在弹出的对话框中点击完成，完成读卡器其中一个驱动安装，如图

 

    （7）、右击我的电脑图标，点击属性弹出系统属性窗口，点击硬件，在硬件对话框中点击设备管理器按钮，弹出设备管理器对话框，找到智能读卡阅读器，展开点击如下图所示的插件名称。添加USB端口步骤与上一个插件相同。
