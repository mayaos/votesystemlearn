1.1 背景

随着社会转型、经济转轨和城市化进程的加快，越来越多的流动人口流向城市。数量巨大的流动人口，在为城市建设、经济发展做出重大贡献的同时也给城市管理带来了巨大的压力。流动人口管理和服务，将会摆脱过去防范式的管理理念，实现以社会控制为主的治安管理模式向城市统筹规划、综合服务管理模式的转变，符合社会化管理服务的发展方向。

东信和平流动人口居所服务管理系统针对政府在流动人口服务管理过程中所遇到的信息采集难、上门服务难、信息质量差、安全没有保障等情况，利用信息化手段，通过建立统一的居所服务管理平台，为政府主管部门更好地进行国土规划、区域规划、城市规划与人口规划提供依据，充分体现政府执政为民的理念。
